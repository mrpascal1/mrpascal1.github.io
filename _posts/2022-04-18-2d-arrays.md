Two Dimensional arrays can be defined as arrays inside an array. the elements in 2D array can be defined as rows and columns.
You can think of 2D arrays as N x M matrix.

<!--more-->
2D arrays are created to implement a relational database lookalike data structure. It provides ease of holding the bulk of data at once which can be passed to any number of functions wherever required.

Some examples of 2D arrays are periodic table, movie rating by multiple reviewers, etc.
<br>

### When you need 2D arrays?
These are some scenarios where 2D arrays are implemented:

*   Digital images.
*   Board games (Chess, Tic Tac Toe, Checkers).
*   Many games use 2D arrays to plot visual environment of a game.

### Creating 1D array vs 2D array?
As a programmer, I beleive you've already came across One-Dimensional arrays. We can create 1D array like:<br>

```python
# In python, we can use list to represent arrays.
# List index also start with 0
one_dim_array = [1, 2, 3, 4]
print(one_dim_array[0]) # Output: 1
```
Now let's see how we can create 2D array:<br>

```python
# In python, we can use list[[],[]] (list of lists) to represent arrays.
two_dim_array = [[4, 5, 6, 7], [8, 9, 12, 1], [10, 11, 3, 2]]

print(two_dim_array[0][0]) # Output: 4
```

Iterate over 2D array:<br>
```python
# In python, we can use list[[]] (list of lists) to represent arrays.
two_dim_array = [[4, 5, 6, 7], [8, 9, 12, 1], [10, 11, 3, 2]]
for i in range(len(two_dim_array)):
    for j in range(len(two_dim_array[i])):
        print(two_dim_array[i][j], end=" ") # Output: 4 5 6 7 8 9 12 1 10 11 3 2
```
